Experiment 6 - Nuclear Catastrophe

A visual representation of the potential catastrophe if a nuclear war was to begin, showcasing humanity dangling on a thin thread where even one small mistake can have disastrous consequences.

Documentation

My inspiration for this experiment drew primarily from the following pieces:

M_6_1_01 by Benedikt Groß, Hartmut Bohnacker, Julia Laub, Claudius Lazzeroni with contributions by Joey Lee and Niels Poldervaart and Bear 71 by Jeremy Mendes and Leanne Allison.

I thought it was interesting to learn about MAD (mutually assured destruction) in classes from long ago, and how close we as a species have gotten close to nuclear warfare numerous times over the last century. I wanted to try to create a visualization of that to show how humanity is essentially living on a delicate thread that is waiting to snap from even a small mistake.

Documentation

I started with M_6_1_01 as it was the artwork that was the most close to resembling what I wanted to achieve with my experiment. I started by changing the color palette with red dots on a black background as I felt they would create a stronger contrast when layered with a white country in between.

As I was working on my code trying to create a randomized “island” for the dots to spawn on, I accidentally created this effect where the the dots “branch” out from the epicenter which I thought was interesting to look at. I then brainstormed an idea of having “red pillars” from the warheads’ points of impact as a way to showcase the sheer magnitude of the circumstances surrounding a nuclear war.

Documentation

I first started by grabbing a free-to-use map of the United States so that I have a canvas to plot my points in. I photoshopped the image to black and white and removed the neighboring countries so that I can focus solely on the United States for data plotting.

I then added the map in a 3D environment so that I can apply a “perspective” when the map will eventually be filled with data points.

I then changed the color of the background to add better contrast and added functionality to allow the user to navigate around the map; clicking and dragging will pan the map, while the scroll wheel will zoom in and out.

Next, I added a way to parse data from a CSV file that contains the state, city, x-coordinate, y-coordinate, and number of warheads for each data point. I also added text at the top to signify what the data is supposed to represent.

I then provided example CSV data to then create a visualization of the data that would best showcase each point in a way that was clear and concise.

After some refinement, I added the dataset to the CSV file which will then be represented on the map as red pillars, the height of which represents the number of warheads that would possibly be sent to that area in a nuclear war.

Finally, I added the option to allow the user to switch between two different visualizations as well as hiding the labels for a clearer view of the map. I felt that a pillar that showed the number of warheads wouldn’t encapsulate the gravity of the circumstance, so I thought having a visual indication of the range of destruction the warheads would cause would serve as an eye-opening realization.

As a final adjustment, I changed the zoom feature from the mouse scroll wheel to the keyboard keys Q and E as I encountered issues where trying to scroll in the canvas would simply scroll the entire browser page instead.

Documentation

  • Contributions

    I made the entirety of the code for this artwork, originally taking inspiration from work done by the inspirers mentioned above.

  • Reflection

    I made a visual representation of the possible destruction of the United States if nuclear war was to occur (according to data sourced from 2015). The user can switch between a numerical and radial visualization of the nuclear destruction to get a grasp of the deadliness of such a battle and of the dreadfulness of a land ravaged by war.

  • Highs

    Even though this experiment took much longer than I anticipated, I still really like what I was able to create with the data represented as “flags” where the height of the flag represents the quantity of warheads that are likely to target the specified location in the United States.

  • Lows

    Creating an experiment that I felt was polished and interesting was much more difficult than I anticipated. In addition, I couldn’t find data that I could easily import into a CSV file, so I had to manually add the data myself which took a substantial amount of time.
